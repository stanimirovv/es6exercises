Underscore.js is a utility-belt library for JavaScript that provides support for the usual functional suspects (each, map, reduce, filter...) without extending any core JavaScript objects.
It is opensource and you can get it here Underscore site

Features:

  1. - Small size
  2. - 100+ functions
  3. - No standard object modification
  4. - Open source
Why use it?
  1. - Faster development time
  2. - Less bugs
  3. - Less lines of code that need to be written
How to install ?
  1. Add from cdn CDN
  2. Or download and include manually. No dependencies.

Hello world in underscore:

_.random(3, 10)

will print a random integer between 3 and 10 inclusive

Useful array functions:

flatten - flattens a nested array

_flatten([1, [2], [3, [[4]]]]

Returned value: [1, 2, 3, [[4]]];

Useful array functions PT2:

uniq - removes duplicates from an array

_.uniq([1, 2, 1, 4, 1, 3]);

Returned value: [1, 2, 4, 3]

Useful object functions:

pick - return copy of object with selected values, easily filter out private fields before returning result

_.pick({publicField: 'asd', publicField2: 'dzhe', privateField: 'secretField'}, 'publicField', 'publicField2');

Returned value: {publicField: 'asd', publicField2: 'dzhe'}

Useful object functions PT2:

isEqual - deep comparison between objects

var stooge = {name: 'moeFoe', luckyNumbers: [13, 27, 34]};
var clone = {name: 'moeFoe', luckyNumbers: [13, 27, 34]};
_.isEqual(stooge, clone);

Returned value: true, but == comparisson would return false
